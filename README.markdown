
# SpeakerBoxx
a lightweight quartz composer patch to playback audio

### HOW TO INSTALL
move SpeakerBoxx.plugin into ~/Library/Graphics/Quartz Composer Plug-Ins/

### NOTES
* the Location may be a file path or a file url
